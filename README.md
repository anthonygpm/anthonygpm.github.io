# Portfólio

## Escopo
Landing page de portfólio pessoal com foco em responsividade, interação e organização visual. Feita para o curso de Programação Full-stack do OxeTech Academy.

## Características do Projeto
- HTML semântico
- Layout responsivo com Tailwind
- Formulário com validação nativa e dinâmica
- Interação com DOM e eventos
- Feedback visual para ação do usuário
- Publicação via GitHub Pages

## Como rodar localmente
1) Abrir o arquivo index.html no navegador
2) (Opcional) Gerar o CSS com Tailwind
   - npm install
   - npx @tailwindcss/cli -i input.css -o styles.css --watch

## Decisões técnicas
- Tailwind v4 CLI para estilos utilitários e responsividade
- JavaScript puro para validação do formulário e feedback
- Estrutura simples com HTML único e assets separados

## Estrutura
- index.html: estrutura da página
- input.css: fonte do Tailwind
- styles.css: CSS gerado
- scripts.js: comportamento e validação
- images/ e fonts/: assets
